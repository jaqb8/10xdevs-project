# Language Learning Buddy v1.6.2

## What's New

- **Keyboard shortcut for analysis submission** - Users can now press `Ctrl + Enter` (Windows/Linux) or `⌘ + Enter` (macOS) to submit text for analysis directly from the textarea, improving the user experience for power users.

## Fixed

- **GitHub workflow changelog path** - Fixed the changelog file path in the master workflow from `docs/v{version}/changelog.md` to the correct `docs/versions/v{version}/changelog.md` location, ensuring release notes are properly included in GitHub releases.

## Changed

- **Analysis form button** - The "Analyze text" button now displays the keyboard shortcut hint (`Ctrl + Enter` or `⌘ + Enter`) to indicate the available shortcut to users.
